Update Instruction

  • Please remove SD card from GPS device before updating map.
  • Connect the GPS device to computer with the Internet connection, and select Mass Storage Mode.
  • Install the file “launcher.exe” to computer and execute “launcher.exe” directly.
  • Follow Garmin Updater instruction. Wait until detecting GPS model name and select OK.
  • Garmin Updater lists updating items, such as map update and software update. Select Update and follow on screen instruction.
  • Wait until Update Completed appears, select Exit. Then update has been completed.
  • Remove the USB connection from computer.
